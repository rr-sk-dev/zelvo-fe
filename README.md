# Zelvo

Angular 22 frontend, styled with Tailwind CSS v4.

## Development

Install dependencies and start the dev server:

```bash
npm install
npm start
```

The app runs at `http://localhost:4200/` and reloads on source changes.

## Building

```bash
npm run build
```

Build artifacts are written to `dist/`. Use `npm run watch` for an incremental development build.
